# Responsive Website Using CSS Media Queries

## Objective

Convert a desktop-only webpage into a mobile-friendly responsive webpage using CSS Media Queries.

## Technologies Used

- HTML5
- CSS3
- Media Queries
- VS Code
- Chrome DevTools

## Features

- Responsive Navigation Menu
- Flexible Layout
- Mobile-Friendly Design
- Scalable Images
- Responsive Cards
- Mobile View Support

## Media Query

```css
@media screen and (max-width:768px)
```

Changes include:

- Navigation stacked vertically
- Cards displayed in one column
- Reduced font sizes
- Responsive spacing
- Images scale automatically

## How to Run

1. Download the repository.
2. Open the project in VS Code.
3. Open index.html in Chrome.
4. Press F12.
5. Toggle Device Toolbar.
6. Select a mobile device.
7. Verify responsive layout.
